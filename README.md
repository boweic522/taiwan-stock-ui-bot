# Taiwan Stock UI Bot

Trader Camp Intelligence — 凱衛風格 Discord 台股查詢機器人

## 指令

| 指令 | 說明 |
|------|------|
| `!股 2330` | 查詢個股即時資訊與 K 線圖 |
| `!查 2330` | 同上 |
| `!k 2330` | 同上 |
| `!help` | 顯示所有指令 |

## 部署

1. 設定 `.env`：填入 `DISCORD_TOKEN`
2. 安裝套件：`pip install -r requirements.txt`
3. 啟動：`python main.py`
